# Flask Portfolio Web App

## Description
This is a multi-page web application built using Python and Flask. 
The project demonstrates routing, template rendering, dynamic content, 
form handling, and CSS styling.

## Features
- Multi-page navigation (Home, About, Resources, Contact)
- Dynamic current date and time
- Contact form with POST request handling
- Custom CSS styling
- Organized Flask project structure

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run Locally

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open browser at:
   http://127.0.0.1:5000/
